# BugBazaar

An application that finds current security vulnerabilities and lists them with a graphical interface. It also shows which countries a vulnerability could most affect. It was made as a project for Istanbul Geli≈üim University Programming Fundamentals course.

## Installation

Use the "python main.py" Bugbazaar will install all packages automatically

```bash
python main.py
```

## Usage
```bash
python main.py
```
